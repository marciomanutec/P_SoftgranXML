# P\_SoftgranXML

Teste Técnico – Importação de XML de NF-e



1 - Configurar o arquivo softgranXML.ini com as informações de conexão com o banco de dados.

2 - O arquivo softgranXML.ini deve estar na mesma pasta do executável do sistema.

3 - Ao abrir o sistema ele carrega a aba de pesquisa mostrando os documentos importados.

4 - Na aba de pesquisa clicando no botão Selecionar é possível selecionar o arquivo xml para importação.

5 - Na aba de pesquisa depois de selecionar o arquivo xml clica no botão Importar.

6 - Na grid de pesquisa seleciona o documentoNFe e clica em Alterar para ir para aba de Dados d NFe.

7 - Na aba de Dados da NFe temos os botões Processar e Reprocessar.
